# Agent Instructions

This project is built on the [Cofound](https://github.com/gilbert/cofound) framework. Before doing any framework-level work — server routes, database schema & migrations, background jobs, sessions, SSR, or CSS — read the Cofound documentation that ships with the installed package:

**`node_modules/cofound/docs/README.md`**

That README is the index for the full guide (schema & migrations, routes, the request object, jobs, sessions, and the frontend APIs). Treat it as required context, not optional reference material.
